Q3) difference between abstraction and interface?
 
  1)Abstract class can have abstract and non-abstract methods.
  1)interface can have only abstract methods. Since Java 8, it can have 
    default and static methods also.

  2)Abstract class doesn't support multiple inheritance.
  2)Interface supports multiple inheritance.

  3) Abstract class can have final, non-final, static and non-static variables.
  3) Interface has only static and final variables

   4) A Java abstract class can have class members like private, protected, etc.
   4) Members of a Java interface are public by default.
